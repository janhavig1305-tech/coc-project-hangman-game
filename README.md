# coc-project-hangman-game
This is the classic word-guessing game. The computer will pick a secret word, and the player must guess it, one letter at a time. The player has a limited number of "lives" (wrong guesses) before they lose.
# Concepts Used
*C: 
- char Array
- for and while loops
- if/else statements
- Functions
*Maths:
- Data Application
- Data Analysis
# How to Compile
gcc hangman.c -o hangman
./hangman
# How to Run
gcc hangman.c -o hangman
./hangman
